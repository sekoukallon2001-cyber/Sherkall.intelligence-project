📋 Supabase SQL Setup

Run this SQL in your Supabase project's SQL Editor to create the leads table:

create table leads ( id uuid default gen_random_uuid() primary key, created_at timestamptz default now(), name text, phone text, email text, interest text, message text, status text default 'new', notes text );

Then in Authentication → Policies, enable Row Level Security and add a policy to allow anonymous inserts (so the website form can submit). The admin dashboard reads using your service key stored locally.
